# Security Policy
At NaChlorid, we want to make sure our code is secure and
safe to run. For this, we need our community's support.

If you find any vulnerabilities in the Civita Discord Bot or cAPI, please report them by sending a detailed description to **security@nachlorid.example** or by opening a private security advisory via the repository's **"Security" â†’ "Report a vulnerability"** feature. Include steps to reproduce, affected versions, and any relevant logs or screenshots so we can investigate and remediate the issue as quickly as possible.
