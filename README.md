# Civita-rewrite
Civita is a "swiss knife" of discord bots made on Python with moderation tools and fun commands


## Why civita?
- Civita is written by hand
- Civita is free and open-source
- Civita doesn't ask you for your social security number or credit card(Hi MEE6 ;) )
- Civita has a publicly accessible API(cAPI) which allows you to get server's data and etc. using code

## Commands
**Fun**
- `/mcjs_status` - Gets info of a minecraft server
- `/coinflip` - Flips a coin(Heads or Tails)
- More later...

**Moderational commands**
- `/ban` - Bans a user
- `/kick` - Kicks a user
- `/unban` - Unbans the user
